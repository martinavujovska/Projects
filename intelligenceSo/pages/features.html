IntelligenceSO is a cutting-edge, web based software system designed to enhance customer's experience. It provides valuable features to save energy, reduce bills, and budget more effectively by offering self-servise optionbs, promoting environmental awareness, and strenghtening the company brand, it's going to foster stronger relationships between IntelligenceSO and its customers.
First release of the software will provide to the IntelligenceSO customers a modern, easy to use, desktop, tablet and mobile based interface that will be able to display various power consumption overviews so that data from different periods can be easily compared and benchmarked to the average power consumption to different geographical locations.
In order to create the data warehouse power consumption and meters database in the most coordinated and cooperative way possible, a combined development team made of talented individuals specially designated by IntelligenceSO and Norliga will collaborate on it. This team will take into account requirements for the IntelligenceSO data warehouse.
When some power consumption alarm is raised up, based on its definition these actions could be performed: sending e-mail to a list of e-mail addresses via SMTP, sending SMS to a list of numbers via SMPP or HTTPS protocol to some SMS providers that expose different REST API's.
It shall be possible for a user to link to a friend's meter data. Could be one on one or in a group, where multiple meters will be linked to it. User should be able to compare savings, consumptions with a friend.
